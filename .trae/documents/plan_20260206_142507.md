# Implementar Breadcrumb Search Expansível

## 1. Criar componente `BreadcrumbSearch`
Local: `components/shared/Breadcrum/BreadcrumbSearch.tsx`

Este componente será responsável por gerenciar a UI de busca expansível.
- **Estado Fechado**: Renderiza um `BreadcrumbActionIcon` com ícone de Lupa.
- **Estado Aberto**: Renderiza um overlay absoluto (`absolute inset-0`) que cobre todo o breadcrumb com um campo de input e botão de fechar.
- **Props**: `value`, `onChange`, `placeholder`.

```tsx
// Rascunho da lógica
if (isOpen) {
  return (
    <div className="absolute inset-0 z-50 flex items-center bg-background px-2 animate-in fade-in slide-in-from-right-10">
      <Search className="w-4 h-4 mr-2 text-muted-foreground" />
      <input autoFocus className="flex-1 bg-transparent outline-none..." />
      <Button onClick={() => setIsOpen(false)}><X /></Button>
    </div>
  )
}
return <BreadcrumbActionIcon icon={Search} onClick={() => setIsOpen(true)} />
```
