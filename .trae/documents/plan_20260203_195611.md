Perfeito. Criar um componente dedicado é a melhor solução para manter o editor de lições leve e focado, sem interferir no editor de documentos complexo que já existe.

### Plano de Implementação

1.  **Criar Novo Componente: `components/course/CourseEditor.tsx`**
    *   Será uma versão otimizada do TipTap para blocos de conteúdo.
    *   **Props:** `content`, `onChange`, `readOnly`.
    *   **Extensões:** `StarterKit` (negrito, listas, títulos), `Link`, `Image`, `Underline`, `TextAlign`.
    *   **Interface:**
        *   **Modo Edição:** Barra de ferramentas simples (Bold, Italic, H1, H2, Listas) + Área de texto com altura automática.
        *   **Modo Leitura:** Apenas o conteúdo renderizado, sem bordas ou menus.

2.  **Atualizar `LessonForm.tsx`**
    *   Substituir o `Textarea` pelo novo `<CourseEditor />`.
    *   Conectar o `onChange` para atualizar o estado do bloco imediatamente.

3.  **Atualizar `LessonDisplay.tsx`**
    *   Substituir `ReactMarkdown` pelo `<CourseEditor readOnly={true} />`.
    *   Isso garantirá que a visualização seja idêntica à edição.

Dessa forma, teremos um editor limpo, rápido e específico para a criação de cursos.
