# Efeitos Sonoros do Jogo

Para habilitar os efeitos sonoros, adicione os seguintes arquivos `.mp3` nesta pasta:

- `correct.mp3` - Tocado quando o jogador acerta uma resposta.
- `wrong.mp3` - Tocado quando o jogador erra.
- `victory.mp3` - Tocado na tela de resultados final.
- `start.mp3` - Tocado quando o jogo inicia.
- `click.mp3` - (Opcional) Tocado ao clicar em botões.
- `countdown.mp3` - (Opcional) Tocado nos últimos segundos. // NÃO VAMOS USAR ESSE

> **Nota:** O sistema está configurado para falhar silenciosamente se os arquivos não existirem, então o jogo funcionará normalmente sem eles.
